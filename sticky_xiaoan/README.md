# 洨洨安的載入動畫
## [Demo](demo.html)
## 這適合用在什麼地方
大型網頁、或是一些讀取夭壽久的鬼網頁，但不推薦用在十秒內能讀完的網頁
## 如何使用
將底下程式碼貼進 HTML 裡就可以ㄌ
```html
<link href="pace_sticy_xiaoan.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/pace/1.0.2/pace.min.js"></script>
```
